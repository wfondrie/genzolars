## What is this?
genzolars is an alternative syntax for polars that replaces boring old function names with GenZ slang. 
Your data wrangling is about to hit different.


## Coding conventions
- The project is managed with uv
- Tests are run with `uv run pytest`
- Linting and formatting are performed with Ruff, `uv run ruff`
- We also have pre-commit hooks, installed with `pre-commit install`
